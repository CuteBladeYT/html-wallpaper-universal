Installation

NOTE: Installation depends on what operating system you use.


• Windows

1. Download and Install WallpaperWebPage application.
2. Run it.
3. On taskbar will appear it's tray icon. Right click it then Configure.
4. A new window with URL input will appear.
In place where the window asks about URL adress, copy the location of wallpaper.html and paste it.
It may look similar to this:
C:/Users/you/Desktop/Wallpaper Universal/wallpaper.html, but remember to always include wallpaper.html in the end!
5. Press OK.
6. Done!


• Linux

This plugin is the only way I found and tested.





Setting Wallpaper

The process is very simple!
1. If you want to set wallpaper as image, make sure it is in .png format.
If it's a gif, it should be a .gif. For video and .mp4 then.
2. Rename it to bg no matter what is after the dot.
3. Move it to wallpapers folder where HTML Wallpaper is located.
4. Refresh wallpaper.
5. Done!





Installing Modules

1. Download module from Github.
2. Unpack it. 3. Copy the folder to modules folder where HTML Wallpaper is located.
4. Open settings.js in parent folder with any code / text editor.
5. Slide down and find WALLPAPER PRELOAD section.
6. Inside the document.addEventListener() { function place loadModule(path)
Replacing path with the name of folder from 3.
7. Refresh wallpaper.
8. Done!
